Presentation of the project

The app is a Flask-based chatbot web application that uses NLP and a deep learning model (trained with Keras and TensorFlow) to interact with users. It classifies user inputs (sentences) into predefined intents and responds accordingly.

Technologies Used

Technologies Used

  • Flask: For the backend server and API handling.
  • HTML, CSS, JavaScript: For the frontend user interface.
  • Deep Learning: Using Keras (with TensorFlow) for model training and inference.
  • NLP (Natural Language Processing): For processing and understanding user input, utilizing libraries like NLTK.

Functionalities

Data Preprocessing

  • Words: Holds all unique words found in the patterns.
  • Classes: Holds all unique intents (tags).
  • Documents: Contains pairs of tokenized patterns and their respective tags.

The preprocessing steps include:

  • Tokenization: User inputs (patterns) are tokenized using nltk.word_tokenize().
  • Lemmatization: Each word is lemmatized (converted to its base form), while ignoring certain characters (such as ?, !, and .).
  • Storage: After preprocessing, the words and classes are sorted and stored in words.pkl and classes.pkl files for later use.

Training Data Creation

  • Bag of Words Representation: For each pattern in the documents, a "bag of words" is created. This is a list where each element indicates whether a specific word (from the list of words) is present in the pattern.
  • Output Row: Represents which class (intent) the current pattern corresponds to, using a one-hot encoded vector.

The training data is stored in a variable called training, which is then shuffled for better training.

Neural Network Model

The output layer of the model uses softmax activation to predict probabilities for each intent, enabling multi-class classification. The model is trained to classify user input into different intents based on the provided patterns and responses in the JSON file. After training, the model is saved to handle future queries.

Developing the Chatbot

I utilize the model.h5 file to develop the Flask application for the chatbot. This file contains the trained deep learning model, which is essential for predicting user intents based on their input. The Flask application serves as the backend, handling requests from the frontend, processing user messages, and returning appropriate responses generated by the model.
