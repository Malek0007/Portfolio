User dashboard with chatbot

Link to GitHub

Presentation of the project

The project is a user dashboard for managing node information, configurations, and details about data centers, including their locations. The dashboard displays a dynamic map and represents information through charts such as curves and pie charts. Another key component of the user dashboard is the chatbot, which assists by answering questions related to node information details or other relevant topics.

Technologies Used

  • MERN: Used for building the full-stack application, with MongoDB for database management, Express and Node.js for backend services, and React for the frontend interface.
  • Generative AI (GPT-4 model): Powers the chatbot that answers questions related to node information and data center configurations.
  • Data visualization (Nivo): Used to create dynamic charts and graphs (such as curves and pie charts) to represent node and data center information.
  • Map display (GeoJSON and React-Leaflet): Utilized for rendering interactive maps that display data center locations and other geographical information.

Functionalities

Data Processing

Before visualizing the data, I first process it to check for any empty fields or errors using the Pandas library. This ensures that the data is clean and ready for visualization.

Application Development

Developed a MERN application with full CRUD (Create, Read, Update, Delete) functionalities. It serves as a dynamic dashboard to facilitate the management of nodes, making it easy to update configurations and monitor data center information.

Data Visualization

I used the Nivo library (@nivo/line and @nivo/pie) to display dynamic charts such as curves and pie charts. The curves represent revenue and profit of nodes by month, while pie charts visualize total sales by category. Additionally, monthly and daily sales are presented through various charts to offer detailed insights into node information.

Chatbot Integration

A chatbot powered by GPT-4 is integrated to answer questions related to node information, data center details, and other relevant topics. This provides an intuitive interface for users to interact with the system and obtain information effortlessly.
