Text Summarizer

Link to GitHub

Presentation of the project

The interface is simple and easy to use. Users can enter their text into a provided box and adjust a slider to set the maximum length of the summary. After clicking the "Submit" button, the summarized text appears right below the original input.

Technologies Used

This project uses the following technologies:

  • Flask is used to build the web interface and handle routing.
  • Hugging Face API powers the text summarization model.
  • HTML/CSS/Javascript is used for building the user interface.

Functionalities

Automatic Text Summarization Model

The app uses Hugging Face's API the BART model, which is designed for summarizing text. The API is called through a POST request, sending the user's text and the chosen summary length (minimum and maximum) in the request.

Processing and Display

Once the user submits their text, the backend sends it to the API, which returns a summarized version. The summary is then displayed on the page alongside the original text for easy comparison.

Backend using Flask

The application is built using Python with the Flask framework. It handles two main routes:

  • The / route displays the homepage with the text input form.
  • The /Summarize route processes the text and sends it to a text summarization model.

Project Usefulness

This project is useful for quickly summarizing long texts like articles, reports, or documents. It is a great example of how web applications can integrate natural language processing (NLP) models to provide efficient solutions for text analysis.
